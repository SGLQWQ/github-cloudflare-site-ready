# 虚拟货币科普指南

这是用于 GitHub Desktop 与 Cloudflare Pages 的静态网站版本。

## Cloudflare Pages 设置

- Production branch：`main`
- Framework preset：`None`
- Build command：`exit 0`
- Build output directory：`public`

完成首次部署后，Cloudflare 会提供一个 `项目名.pages.dev` 访问链接。

## 更新网站

在 GitHub Desktop 中提交修改并点击 `Push origin`。Cloudflare Pages 会在检测到 `main` 分支的新提交后自动重新部署。
